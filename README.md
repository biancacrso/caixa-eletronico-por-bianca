# Sistema de Caixa Eletrônico em PHP desenvolvido por mim, Bianca Pereira Cardoso, do 3° período de Sistemas de Informação.

Aplicação em PHP que simula um caixa eletrônico, desenvolvida conforme atividade proposta em sala de aula.

**Funcionalidades:**

- Depósito
- Saque (com validação de saldo)
- Visualização de saldo

**Tecnologias:**

- PHP
- HTML
- CSS
- XAMPP
