# Sistema de Caixa Eletrônico em PHP criado por Bianca Pereira Cardoso, estudante do curso de Sistemas de Informação do 3° período.

Aplicação em PHP que simula um caixa eletrônico, desenvolvida conforme atividade proposta em sala de aula.

**Funcionalidades:**

- Depósito
- Saque (com validação de saldo)
- Visualização de saldo

**Tecnologias:**

- PHP
- HTML
- CSS
- XAMPP
